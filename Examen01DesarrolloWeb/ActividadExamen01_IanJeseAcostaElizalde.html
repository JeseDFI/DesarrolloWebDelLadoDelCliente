Tecnologías: conceptos clave

Una visión técnica y organizada de tecnologías relacionadas con la web y redes.

Introducción

Las tecnologías que sustentan la web incluyen protocolos de comunicación (como TCP/IP), servicios de hospedaje (hosting), protocolos de transferencia de archivos (FTP), y tecnologías de desarrollo (HTML, CSS, JavaScript).

Redes y protocolos

Las redes permiten la interconexión de dispositivos. Protocolos como IP proporcionan direccionamiento, mientras que TCP asegura la entrega fiable de paquetes. ARPANET (1969) fue el antecedente histórico de Internet.

Hospedaje web

El hospedaje web es un servicio de infraestructura para almacenar y servir sitios mediante servidores conectados a Internet. Tipos comunes: compartido, VPS, dedicado y en la nube.

FTP y transferencia de archivos

FTP (File Transfer Protocol) es un protocolo para transferir archivos entre cliente y servidor. Se utiliza habitualmente para publicar archivos en servidores web o sincronizar contenido.

Desarrollo web

HTML5 ofrece elementos semánticos (como los que se usan en esta página) que mejoran la accesibilidad y el SEO. CSS y JavaScript se encargan del estilo y la interactividad, respectivamente.
